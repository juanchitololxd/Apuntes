Big data


Grandes volumnes de datos muy variados


Resulta complicado procesarlos de manera tradicionales (en una PC o en servidores ordinarios)




Para que sea big data debe cumplir las 5V


5V


Volumen

Almacenamiento masivo de grandes cantidades de datos de multiples fuentes (puede ser en vivo desde redes sociales)

Velocidad

Los datos deben ser procesados a la misma velocidad que se generan

Variedad

Distintos tipos de datos, por ej: Tablas, JSON, imagenes, videos etc.

Veracidad

Calidad y confiabilidad de los datos

Valor

Los datos deben poder proporcionar insights



Procesamiento


Se procesa al dividirla en partes mas pequeñas en distintos servidores


Se usan tecnologías de apache como spark, headoop en conjunto con servicios de computo en la n




Tipos de datos


- De personas: Se analizan datos en base a las acciones de las personas, como al dar like en Fb

- De transacciones Monetarias y no monetarias

- De navegación web Ej: Cookies

- Biométricos

- Machine 2 machine Datos con los que se comunican 2 aplicaciones, como el GPS y UBER




Cual es la diferencia entre Tipo de dato de personas y de transacciones no monetarias?

Que la segunda solo permite medir como se siente el usuario usando el servicio, mientras que con el primero se puede conocer a la persoan mas en específico
