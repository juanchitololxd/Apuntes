General


Los puestos de data science estan en cualquier industria y pagan desde $1000


Glosario


Algoritmo

conjunto de pasos ordenados que en conjunto dan solución a un problema

Algoritmo de machine learning

Permiten generar modelos entrenados con los que se puede predecir información

API

Application program interface. mecanismo que permite a dos componentes de software comunicarse entre sí mediante un conjunto de definiciones y protocolos

Backend

la capa de desarrollo web donde se ejecutan procesos de acceso a datos de una aplicación web.


Bases de datos

Es el conjunto de datos pertenecientes a un mismo contexto que son almacenados para su uso en algún sistema de software.

Bases de datos OLAP

bases de datos especializadas para generar consultas

agilizadas de grandes cantidades de datos. (A: Analytics)


Bases de datos OLTP

BD que se enfocan al registro y actualización de datos (T: transacciones).


Computo en la nube

Son los servicios en la nube para almacenar, administrar y procesar

datos, servidores, bases de datos, redes y software.


Computo paralelo

Muchas instrucciones se ejecutan en paralelo

Data pipeline


Proceso donde, con diferentes tecnologías, se mueven y procesan datos.

Usado para mover datos de BD OLTP a BD OLAP


Datos

Registros que se almacenan en documentos informáticos

ETL

Extract, Transform, Load. Es un tipo de pipeline donde se extraen datos de distintas fuentes, se transforman y se cargan en bases de datos especializadas para analítica


Front-end

Parte de aplicación donde se crea la interfaz para el usuario

Insights o Información de valor

Es la info que se ha obtenido luego de analizar datos



Limpieza de datos

se refiere a cuando se descubren datos erróneos o

faltantes en una tabla o base de datos y se corrigen o eliminan

para poder analizarlos.


Machine learning

Es una rama de la IA

Busca que las maquinas de cómputo aprendan observando grandes cantidades de datos


Modelo de machine

salida de información que se genera cuando se entrena un

algoritmo de machine learning con datos.


Transformación de datos

Proceso que implica separar datos, limpiar datos nulos, agregar nuevas columnas con nueva información, e incluso cambiarles el

formato.
