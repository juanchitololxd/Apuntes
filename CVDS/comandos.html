Comando Uso Extra
mvn archetype:generate -DgroupId=edu.eci.cvds -DartifactId=patterns -DarchetypeArtifactId={maven-archetype-quickstart | maven-archetype-webapp} Crear proyecto con el Arquetipo indicado
mvn clean Borra los archivos generados por mvn package
mvn exec:java -Dexec.mainClass="{path}" -Dexec.args="{arg1 'arg 2' arg3 arg4 ...}" Ejecutar el archivo
  • path es el path desde src hasta la clase main. ejemplo: "edu.eci.cvds.patterns.App"
mvn test -Dtest="{path.to.test}" Ejecutar test "edu.eci.cvds.patterns.AppTest"
