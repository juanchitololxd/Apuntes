Diferencia entre sistema centralizado y distribuido?

GIT

Que es HEAD?

Es el commit sobre el que estoy trabajando

Si se escribe git checkout main^ que se quiere decir?

Poner '^' luego del nombre de la rama es equivalente a decir "el padre de la rama
Si se usa '~n' en vez de '^', se puede navegar n-niveles especificados
Ejercicio Cool Rta
Mover forzosamente la rama main tres padres por detras de HEAD git branch -f HEAD~3 ó git branch -f main {hashCommit}
Comando Que hace ejemplo
git rebase {rama} Pone el commit de la actual rama por encima del de {rama} git rebase main
git reset {hashCommit o HEAD~{n}} Borra los commits realizados y devuelve el HEAD a donde se indique. Para que funcione En remoto se debe escribir git revert {hashCommit o HEAD~{n}} git reset HEAD~2
git cherry-pick {hashCommit1} {hashCommit2} ... {hashCommitn} Copia la infomarción de los commits sobre la rama actual git cherry pick as324 acd23
git tag {nombreTag} {hashCommit}
