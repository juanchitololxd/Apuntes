Mejores practicas


No poner el verbo en la URI: `/getCars`

No usar GET para modificar algo

Usar plural, es decir: `cars`en lugar de `car`

Usar subrecursos: `GET /cars/711/drivers/ Returns a list of drivers for car 711 GET /cars/711/drivers/4 Returns driver #4 for car 711`

Usar Http-Headers para serialización de formatos, tanto para aceptar el request como pa aceptar el response

Usar HateOAS

Versionar la API

Usar Http Status codes
