Atributos de calidad


Propiedad medible de un sistema que indica qué tan bien se satisfacen las necesidades


Compatibility

IOS, MAC, Windows

Usability

Accesibilidad, protección contra errores de usuario

Security

Implica integridad de datos, confidencialidad etc.

Maintaniblity

Implica que el codigo sea reusable, modularizado, comentareado etc.

Portability

Permite mover facilmente el producto por distintos servers. Se usa docker
