Una buena practica es tener virtual environments, por lo que se crea requirements.txt, aqui se le indica lo que debe tener instalado el proyecto. Para instalarlo todo de un tirón: `pip install -r requirements.txt `








````

mkdir proyecto

cd proyecto

virtualenv venv --python=3.7

venv\Scripts\activate # activar virtualenv

pip install -r .\requirements.txt

$env:FLASK_APP="main.py" # indicarle el archivo que correrá inicialmente

$env:FLASK_DEBUG="1"

````
